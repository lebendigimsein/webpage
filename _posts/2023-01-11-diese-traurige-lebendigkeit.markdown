Auf einmal
Diese Lebensfreude
So viel
So viel
So viel
Das ich umarmen möchte
Mich bewegt
Nährt
Erfrischt
Zum Lachen bringt
Zu inneren Purzelbäumen
Und zugleich
Ganz tief
Lebt eine tiefe Traurigkeit
Diese Schwere
Wartet
Auf die Zeit
Sich ganz zu zeigen
Aus tiefstem Herzen
Erkannt zu werden
Umarmt
Lässt mich weinen
So viel
So oft
So lange
Dass all die Freude
Noch etwas warten muss
Mit dem Herausbrechen
